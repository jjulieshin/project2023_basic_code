The Grid System


The CSS Grid Layout Module provides a grid-based layout system with rows and columns, making it easier to design web pages without relying on floats and positioning. (It is a powerful tool for implementing grid-based layouts in your web design) A grid system is an effective way to organize content and create clear pages that are easy for website users to process. You can decide whether to use a 12-column grid layout for your entire site or divide specific sections into multiple areas. The grid is also easy to resize.

We have 4 sizes of grid layout, Mobile(0-599px), mobile wide(600-899px), tablet(900-1199px) and desktop(1200px~)
