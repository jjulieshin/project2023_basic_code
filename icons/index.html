Icons

We are using FontAWesome as our variety features. A link to Font Awesome is included on our Main home page, so start adding icons from our library as much as needed. Simply add the CDN to the <head></head> section of your webpage and add the icon font tags ><i> to your elements. Here are are few that we recommend.

                    
    <head>
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">
    </head>
